# 4️⃣ 프로젝트 후기

- css와 scss 둘 다 적용해 봤는데 css가 익숙해서 그런지 scss의 장점을 모르겠음
- VSC에서 제공하는 확장자 도구로 ESLint(js 코드 분석 도구) 사용이 어려웠음
- 기능 구현이 미숙하여 서비스 완성도가 낮은 점이 아쉬움
